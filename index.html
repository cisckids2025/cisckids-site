How to install:
  1. Download the RAR file to your device.
  2. Extract the RAR file using a file manager app (RAR or ZArchiver from Google Play) to get the APK file.
  3. Open the extracted APK file on your Android device.
  4. If prompted, allow installation from unknown sources.
Tip: Most Android phones can't open RAR files by default.
If you can't extract it, install a free app like RAR or ZArchiver from the Play Store.
